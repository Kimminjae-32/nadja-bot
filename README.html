나쟈 봇

이터널 리턴 전용 디스코드 구인 봇

Discord.js v14 Node.js 18+ Beta

✨ 주요 기능

🎮
파티 구인
일반 / 랭크 / 내전(커스텀) 유형 선택, 맵 선택, 시작 시간 설정
📌
팀 고정
1팀 / 2팀 고정 버튼으로 팀 섞기 시 원하는 자리에 고정 배치
🎲
팀 섞기
고정 인원을 제외한 나머지 대기 명단을 랜덤으로 팀 배정
👑
방장 양도
선택 메뉴로 다른 참가자에게 모집자 권한 이전 가능
⏰
자동 삭제
시작 시간 기준 설정한 시간(기본 4시간) 후 구인 게시글 자동 삭제
🔔
DM 알림
새로운 참가자 발생 시 모집자에게 DM으로 실시간 알림 전송

🚀 명령어

/구인 이터널 리턴 구인을 시작합니다
옵션 필수 여부 설명
유형 필수 일반 / 랭크 / 내전(커스텀)
맵 필수 루미아 섬 / 코발트
랭크 선택 시 자동으로 루미아 섬
시간 선택 시작 시간 (예: 22시 30분)
미입력 시 즉시
종료시간 선택 자동 삭제까지 대기 시간 (숫자)
기본 4시간
/사용법 나쟈 봇 사용 가이드를 확인합니다 (본인에게만 보임)
인원 제한 — 코발트 최대 4인 · 루미아 섬 최대 3인 · 내전(커스텀) 최대 8인

🔧 설치 및 실행

1
저장소 클론
git clone https://github.com/Kimminjae-32/nadja-bot.git
cd nadja-bot
2
패키지 설치
npm install
3
환경변수 설정

프로젝트 루트에 .env 파일을 생성하세요.

TOKEN = 디스코드 봇 토큰
CLIENT_ID = 봇 애플리케이션 ID
4
슬래시 명령어 등록
node deploy-commands.js
5
봇 실행
node index.js

📁 프로젝트 구조

nadja-bot/
├── index.js            # 메인 봇 파일
├── deploy-commands.js  # 슬래시 명령어 등록
├── recruits.json       # 구인 데이터 저장 (자동 생성)
├── package.json
├── .gitignore
└── .env               # 환경변수 (업로드 금지)

⚠️ 주의사항

⚠️ .env 파일은 절대 GitHub에 업로드하지 마세요. 봇 토큰이 외부에 노출됩니다.
⚠️ 봇에게 메시지 읽기/쓰기, 멤버 조회, DM 전송 권한이 필요합니다.
⚠️ recruits.json은 봇 실행 시 자동으로 생성됩니다. 직접 만들 필요 없어요.
