# 🎮 나쟈 봇 (Nadja Bot)

이터널 리턴 내전/구인 전용 디스코드 봇입니다.

---

## ✨ 주요 기능

### ⚔️ 구인
- 일반/랭크 게임 구인 메시지 생성
- 참가/취소 버튼으로 간편 참가
- 인원 달성 시 방장에게 DM 알림
- 구인 메시지 자동 삭제 (기본 24시간)

### 🏟️ 내전
| 커맨드 | 설명 |
|---|---|
| `/내전 루미아 섬` | 팀당 인원/최대인원 자유 설정 |
| `/내전 론울프` | 1인 1팀 개인전 (최대 18명) |
| `/내전 코발트` | 4vs4 고정 |

### 🎮 내전 버튼 기능
| 버튼 | 설명 | 권한 |
|---|---|---|
| 참가 / 취소 | 구인 참가 및 취소 | 누구나 |
| 팀 섞기(자동) | 참가자 랜덤 팀 배정 | 방장 |
| 팀 설정(수동) | 1팀 멤버 직접 선택 | 방장 |
| 실험체 랜덤 | 팀원별 실험체 랜덤 배정 | 방장 |
| 참가자 킥 | 특정 참가자 제외 | 방장 |
| 맵 변경 | 루미아/코발트/론울프 전환 (인원 유지) | 방장 |
| 방장 양도 | 다른 참가자에게 방장 권한 이동 | 방장 |
| 방 이동 | 팀별 음성 채널 이동 | 방장 |
| 원래대로 | 방 이동 전 채널로 복구 | 방장 |

### 🔍 전적 조회 (API 키 필요)
| 커맨드 | 설명 |
|---|---|
| `/전적 닉네임` | 솔로/듀오/스쿼드 전적 조회 |
| `/추천실험체 닉네임` | 가장 많이 쓴 실험체 TOP3 |
| `/랭킹` | 서버 내 MMR 순위표 |
| `/내전결과 승팀` | 내전 결과 기록 |

## 🗺️ 내전 유형별 상세

### 루미아 섬
```
/내전 유형:루미아 섬 팀당인원:3 최대인원:24
→ 8팀 (3인팀 x8), 최대 24명
→ 나누어 떨어지지 않으면 처리 방식 선택 가능
```

### 론울프 (개인전)
```
/내전 유형:론울프 최대인원:18
→ 1인 1팀, 최대 18명
→ 실험체 랜덤 배정 지원
```

### 코발트
```
/내전 유형:코발트
→ 4vs4 고정 (8명)
```

## 🛠️ 사용 기술

- **Node.js**
- **discord.js** v14
- **node-cron** (자동 삭제 스케줄러)
- **Eternal Return Open API**

---

## 📝 참고사항

- `/랭킹` 명령어는 디스코드 닉네임 = 이터널 리턴 닉네임인 유저만 조회됩니다
- 구인 메시지는 기본 24시간 후 자동 삭제됩니다 (종료시간 옵션으로 변경 가능)
- 방 이동 기능은 음성 채널에 접속한 상태에서만 사용 가능합니다
