# 🎮 나쟈 봇 (Nadja Bot)

이터널 리턴 내전/구인 전용 디스코드 봇입니다.

---

## ✨ 주요 기능

### ⚔️ 구인
- 일반/랭크 게임 구인 메시지 생성
- 참가/취소 버튼으로 간편 참가
- 인원 달성 시 방장에게 DM 알림
- 구인 메시지 자동 삭제 (기본 24시간)

### 🏟️ 내전
| 커맨드 | 설명 |
|---|---|
| `/내전 루미아 섬` | 팀당 인원/최대인원 자유 설정 |
| `/내전 론울프` | 1인 1팀 개인전 (최대 18명) |
| `/내전 코발트` | 4vs4 고정 |

### 📋 내전 웹 폼 참가 신청

내전 생성 시 자동으로 **웹 폼 참가 링크**가 함께 제공됩니다.

- 링크 클릭 → 웹 페이지에서 **디스코드 닉네임 / 인게임 닉네임 / 선호 포지션** 입력 후 신청
- 신청 완료 후 **수정 / 취소 링크** 개별 제공
- 방장이 **팀 섞기(자동)** 클릭 시 웹 폼 참가자가 있으면 해당 목록으로 팀 배정 (포지션 정보 포함 표시)
- 웹 폼 참가자가 없으면 Discord 버튼으로 참가한 인원으로 기존 방식 섞기

**선호 포지션**: 탱커 / 전사 / 암살자 / 스킬 딜러 / 원거리 딜러 / 지원가

### 🎮 내전 버튼 기능

**항상 표시되는 버튼**

| 버튼 | 설명 | 권한 |
|---|---|---|
| 참가 / 취소 | 구인 참가 및 취소 | 누구나 |
| 팀 섞기(자동) | 참가자 랜덤 팀 배정 (웹 폼 우선) | 방장 |
| 실험체 랜덤 | 팀원별 실험체 랜덤 배정 | 방장 |
| ⚙️ 관리 | 방장 전용 관리 메뉴 열기 | 방장 |
| 🔗 웹 폼으로 참가 | 웹 참가 신청 링크 열기 | 누구나 |

**⚙️ 관리 메뉴 (내전)**

| 항목 | 설명 |
|---|---|
| 팀 설정(수동) | 1팀 멤버 직접 선택 |
| 참가자 킥 | 특정 참가자 제외 |
| 방장 양도 | 다른 참가자에게 방장 권한 이동 |
| 방 이동 | 팀별 음성 채널 이동 |
| 원래대로 | 방 이동 전 채널로 복구 |
| 맵 변경 | 루미아/코발트/론울프 전환 (인원 유지) |

### 🗓️ 시즌 정보
| 커맨드 | 설명 |
|---|---|
| `/시즌` | 현재 시즌 이름, 시작일, 종료일, 남은 기간 표시 |
| `/무료실험체` | 이번 주 모드별 무료 실험체 목록 표시 |

## 🗺️ 내전 유형별 상세

### 루미아 섬
```
/내전 유형:루미아 섬 팀당인원:3 최대인원:24
→ 8팀 (3인팀 x8), 최대 24명
→ 나누어 떨어지지 않으면 처리 방식 선택 가능
```

### 론울프 (개인전)
```
/내전 유형:론울프 최대인원:18
→ 1인 1팀, 최대 18명
→ 실험체 랜덤 배정 지원
```

### 코발트
```
/내전 유형:코발트
→ 4vs4 고정 (8명)
```

## 🛠️ 사용 기술

- **Node.js**
- **discord.js** v14
- **express** (웹 폼 서버)
- **better-sqlite3** (참가 신청 데이터 저장)
- **node-cron** (자동 삭제 스케줄러)
- **Eternal Return Open API**

---

## 📝 참고사항

- 구인 메시지는 기본 24시간 후 자동 삭제됩니다 (종료시간 옵션으로 변경 가능)
- 방 이동 기능은 음성 채널에 접속한 상태에서만 사용 가능합니다
- 웹 폼 참가 기능은 서버에 `WEB_URL` 환경 변수 설정 필요 (`.env`에 `WEB_URL=http://서버IP:3000` 추가)
